# Ozzy
Лимонады Ozzy • GLB‑превью упаковки (private repo, view‑only).

## Ozzy Beverage Packaging · GLB Previews
Частная коллекция *просмотровых* 3D‑моделей тары лимонадов **Ozzy**  
в формате **.glb** для согласования с клиентами.

> **Важно:** репозиторий закрытый (Private).  
> Файлы предназначены только для просмотра и обсуждения.  
> Скачивание и перераспространение без письменного разрешения запрещены.

### Содержимое
- `/previews` — оптимизированные GLB‑модели (LOD 0, ≤2 МБ каждая)  
- `/screenshots` — PNG‑превью, отображаемые в Issues и письмах  
- `/docs` — PDF‑чертежи габаритов (при необходимости)

### Как смотреть
1. Откройте любой `.glb` прямо в GitHub — встроенный 3D‑viewer позволяет
   вращать модель и включать wireframe.  
2. При необходимости переключитесь в fullscreen (`F`).  
3. Не скачивайте файл, если в этом нет прямой необходимости.

### Контроль доступа
- **Private repo** — ссылка работает только для приглашённых аккаунтов.  
- При необходимости пригласите нового пользователя через  
  *Settings → Manage access* → *Invite a collaborator*.  
- Убедитесь, что у приглашённого стоит роль **Read**.

### Обратная связь
Используйте Issues для комментариев и правок.  
Каждая модель сопровождается скриншотом и ID партии напитка.

#### Советы по настройке приватного репозитория

| Что | Зачем | Как |
|-----|-------|-----|
| **Git LFS** | Хранить GLB без раздувания репо | `git lfs track "*.glb"` |
| **Protected Branch** | Избежать случайных пушей | Settings → Branches |
| **Disallow Forks** | Запретить копии | Settings → General → *Allow forking* (снять галку) |
| **Required Review** | Контроль изменений | Settings → Branch protection rules |
| **Watermark внутри GLB** | Маркер утечки | Малозаметный emboss с датой |

Так README остаётся компактным, но сразу расставляет границы использования и поясняет, как заказчику «просто посмотреть».
